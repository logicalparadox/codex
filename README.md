# Codex

Static site generation from markdown files.

* Content is written in Markdown
* Templates are writen in Jade
* Styles are written in Stylus

Yeah, super easy :D

## Starting Up

```sh
npm install codex -g
codex skeleton new-project
cd new-project
codex build
```
